## 🎯 Sobre o Projeto

Projeto Focado na criação de Pipelines ETL para a verificação de cancelamento dos clientes (churn). o objetivo principal desse projeto é demonstrar as aplicações das minhas skills em

- Criação de Data Bases e tabelas utilizando **SQL**
- Construção de PipeLines de dados **PYTHON**
- Transformação e tratamento de dados **limpeza, feature engineering**

Possivelmente no futuro terá sistemas mais complexos como **airflow** para automatização desse processo e 

**Em breve:** Este projeto será evoluído com a adição de **Apache Airflow** para orquestração, **dbt** para transformações profissionais e **migração para a nuvem (AWS/GCP)**, tornando o pipeline escalável e pronto para produção.

## Tecnologia utilizadas
- **SQL SERVER** - Criação e modelagem de um Data Base 
- **PYTHON** - Criações de Pipelines ETL 
- **PANDAS** - Filtragem e Tratamento de Dados
- **pyODBC** - Conexão Python com SQL Server
